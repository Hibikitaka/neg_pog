# 本テンプレートファイルについて
本テンプレートファイルは、岡山理科大学 工学部 情報工学科 片山研究室専用LaTeXテンプレートです。
尚、本テンプレートファイルの作成にあたっては、原本が紛失していたため、18生の伊東先輩のデータを基に作成した。
卒業論文を書く際は、本ファイルに準じて記述をします。

## 内包ファイル群

テンプレートに含まれるファイル, ディレクトリを説明。

### [dir] ReadMe&InstallGuide
本ファイルが置かれているディレクトリ。さすがにReadMeって書いてあったらまず見るよね？<br>
また、Visual Studio Codeで編集, ビルドする際の環境構築の方法が書かれたPDFファイルを配置している。

### [dir] n_*
数字_* の形式で命名されたファイルは、論文における各セクション毎のTeXファイル "main.tex" と関連するソースが含まれる。<br>
尚、ディレクトリの数字は

* 1～：表紙～実験
* 51～：結論, 謝辞
* 91～：付録

として付番している。


### main.tex
これはセクションディレクトリの main.tex とは違い、本テンプレートの最上位ディレクトリの直下に配置されている。これはセクションディレクトリの main.tex を束ね、１つのTeXファイルとして出力するためのファイルであり、TeXファイルのコンパイル, ビルドはこのファイルによって行う。

### bib-DARP.bib
bibTeXという、参考文献, 引用を司るシステム用のファイル。参考文献はここにリストアップし、TeXファイルから参照する。

### ***.sty

TeXコンパイル時に必要になるスタイルファイル。見た目を記述している。
原則編集しない。

### latexmkrc
ビルド用の様々な設定が記述されている。基本触らなくてOK

### Spine.tex
バインダー背表紙用のTeXファイル。論文が完成するまで気にしなくていい。

### その他
ここで説明されていないファイルは基本気にかけなくても大丈夫(消えたところで再生成される...ハズ？)
