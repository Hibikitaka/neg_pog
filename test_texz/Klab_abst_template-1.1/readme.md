# 片山研究室$\LaTeX$テンプレート
本テンプレートは，片山研究室のアブストラクト用\LaTeX テンプレートである．
中間発表，及び卒業論文発表や修士論文発表時のアブストラクト(要旨)を記述するために使用する．

## 内容物

### ディレクトリ
- fig
  - 画像を格納するディレクトリ．
  - 使わなくてもいいが，プロジェクトルートが汚くなるので配置している
- .vscode
  - vscode用の設定ファイルがある．
 

### ファイル
- main.tex
  - メインのtexファイル．ここに記述する．
- klabAbst.cls
  - スタイル定義用のクラスファイル．
  - **原則編集しないこと**
- .gitignore
  - Gitで管理する際に，不要なファイルを除外するための設定ファイル．
  - Gitを使えるようになろう．
- settings.json
  - vscode設定用ファイル
  - これのお陰でディレクトリを開くだけでコンパイル設定などが読み込まれる．
  - こだわりのある人以外は編集しないが吉．
- readme.md
  - このファイル．
  - まず，最初に，何よりも先に，readmeを読むクセをつけましょう．

## 使用方法

- main.texがあるディレクトリをプロジェクトルートとしてVisual Studio Codeで開く．
- VScode拡張機能「LaTeX Workshop」をインストール．(インストール済みの場合はスキップ)
- main.texをコンパイルし，main.pdfが生成されるか確認する．
- コンパイルされない場合，ビルドレシピがklab_abstbuildを使っているか確認

論文と違い，ページ数は原則少ないため，main.texを保存するたびにコンパイルされる設定になっている．
変更時にコンパイルしないようにするには，.vscode/settings.json内の，`latex-workshop.latex.autoBuild.run`の値を，`never`に変更する．
